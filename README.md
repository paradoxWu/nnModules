# nnModules

realize some nn modules

| 领域                   | 大概率让写                                          | 目的                        |
| ---------------------- | --------------------------------------------------- | --------------------------- |
| **基础 ML**      | K-means、感知机、SoftMax、NLLLoss                   | 语法+向量思维               |
| **CNN**          | 一层 2D Conv + ReLU + MaxPool，残差（ResNet）       | 形状、padding、stride       |
| **RNN/LSTM**     | 一个时间步的 cell（遗忘门、输出门）                 | 循环结构+门控思想           |
| **Transformer**  | Self-Attention 完整公式 + 形状推导                  | 核心 QK^T、softmax、dropout |
| **PyTorch 基础** | Dataset/DataLoader、training step、scheduler.step() | 工程闭环                    |
| **经典 trick**   | BatchNorm/LayerNorm、残差跳连、masked\_fill         | 实现细节+动机               |
| **后处理**       | NMS                                                 |                             |
